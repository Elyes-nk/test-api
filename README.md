# 📝 🕊 Pigeon Back
*Back end of a chat mobile app created by `Elyes Nait kaci.`*

## Technos
- Node Js
- Express Js
- Mongoose
- JsonWebToken
- Crypto-js

## Infos
- MongoDb serveur 
- Hosted in Heroku
- Front end : [https://github.com/Elyes-nk/pigeon]

# Project content
## Authentification

- Login
- Register

## Features 
- Users
- Discussions
- Messages
- Stories
